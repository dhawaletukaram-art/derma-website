# Dermatiqua - Skin, Hair & Laser Clinic Website

🌐 **Live Website:** [https://derma-authority-ready-main.vercel.app](https://derma-authority-ready-main.vercel.app)

## About
Professional dermatology clinic website for Dr. Pradnya Asutkar's practice in Nagpur, India.

### Features
- ✨ Modern, responsive design
- 📱 Mobile-friendly interface
- 🏥 Complete service information
- 📅 Online appointment booking
- 📍 Two clinic locations (Ramdaspeth & Besa)
- ⭐ Patient testimonials and Google reviews
- 🖼️ Treatment gallery
- 📞 Direct contact options (Call, WhatsApp)

## Clinics
- **Dermatiqua** - Ramdaspeth, Nagpur
- **La Aesthetique** - Besa, Nagpur

## Tech Stack
- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/ui
- **Routing:** React Router
- **Deployment:** Vercel (Auto-deploy on push)

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment
Automatically deployed to Vercel on every push to the `main` branch.

**Production URL:** https://derma-authority-ready-main.vercel.app
