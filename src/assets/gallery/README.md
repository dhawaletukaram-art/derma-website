## Image Specifications:
- Format: JPG or PNG
- Size: 600x600px (square format recommended)
- Quality: High resolution, well-lit
- File size: Under 500KB each (compressed)

## IMPORTANT - Patient Privacy:
- ✅ Ensure written patient consent for all images
- ✅ Consider blurring faces if patients prefer
- ✅ Remove any identifying information
- ✅ Keep consent forms on file

## After Adding Images:
1. Update the imports in `src/pages/Gallery.tsx`
2. Replace the placeholder image URLs with your imports
3. Test locally with `npm run dev`
4. Commit and push to deploy

See **IMAGE-REPLACEMENT-GUIDE.md** in the root folder for detailed instructions.
