# Clinic Facility Images

Place your 6 clinic facility photos here.

## Required Images:

1. **clinic-reception.jpg** - Reception & Waiting Area
2. **clinic-consultation.jpg** - Consultation Room
3. **clinic-treatment.jpg** - Treatment Room
4. **clinic-laser.jpg** - Laser Equipment
5. **clinic-procedure.jpg** - Procedure Room
6. **clinic-analysis.jpg** - Skin Analysis Station

## Image Specifications:
- Format: JPG or PNG
- Size: 800x600px (recommended)
- Quality: High resolution, well-lit
- File size: Under 500KB each (compressed)

## After Adding Images:
1. Update the imports in `src/pages/About.tsx`
2. Replace the placeholder image URLs with your imports
3. Test locally with `npm run dev`
4. Commit and push to deploy

See **IMAGE-REPLACEMENT-GUIDE.md** in the root folder for detailed instructions.
